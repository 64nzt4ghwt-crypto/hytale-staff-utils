# StaffUtils for Hytale

Essential tools for server staff and moderators.

## Commands
| Command | Description |
|---------|-------------|
| `/vanish` | Toggle invisibility to regular players |
| `/spy` | Toggle chat spy (see all private messages) |
| `/freeze <player>` | Prevent a player from moving |
| `/warn <player> <reason>` | Issue a formal warning |
| `/warns [player]` | View warning history |
| `/staffchat` | Toggle staff-only chat channel |
| `/stafflist` | List online staff in staff chat |
| `/modmode` | Toggle all staff tools at once |

## Features
- **Vanish** — staff invisible to regular players
- **Freeze** — immobilize suspected rule-breakers
- **Warning system** — persistent warning history with timestamps
- **Staff chat** — private communication channel for staff
- **Spy mode** — see all private messages
- **Mod mode** — enable/disable all tools with one command
- **Data persistence** — warnings saved across restarts
